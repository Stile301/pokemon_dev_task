# Pokemon dev task

## Install yarn
`npm install --global yarn
`

## Install vue CLI
`npm install -g @vue/cli`
or
`yarn global add @vue/cli`

## Upgrade vue CLI

`npm update -g @vue/cli`
or
`yarn global upgrade --latest @vue/cli`

Go to project and open PokemonApp folder in your comand line.

### Install packages
`npm install`
or
`yarn`

### Run project

`yarn dev`
or
`npm run dev`
